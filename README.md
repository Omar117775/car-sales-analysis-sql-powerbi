# Car Sales Analysis | SQL Server + Power BI

## Project Overview

This project analyzes car sales, customers, employees, branches, and services using SQL Server and Power BI.

## Tools Used

- SQL Server
- Power BI
- GitHub

## Database Structure

Tables:

- Branches
- Employees
- Customers
- CarCategories
- Cars
- Sales
- SaleDetails
- Services

## SQL Tasks

- Database Design
- Relationships
- Data Insertion
- Aggregations
- Joins
- KPI Calculations
- Views Creation

## Power BI Dashboard

KPIs:

- Total Revenue
- Total Customers
- Total Cars Sold
- Total Service Revenue

Visualizations:

- Revenue by Branch
- Service Cost Analysis
- Customer Analysis
- Sales Distribution
